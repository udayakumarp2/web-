Semantic Mark Up

Semantic mark up is one the main features of HTML5. It helps with the layout of a normal html page. for example the article, aside, details, figcaption,figure,footer,header, main, mark, nav, section, summary , and time are common semantic elements. It helps plan your page layout when you are restricted to html5 and css alone for an assignment.

Section 508 Accessiblity

Section 508 requires that federal agencies make their technology websites and tools available to everyone this included individuals with disablities. It requires all federal agencies to accomadate people with dissablities. This includes telephones, smart phones and mobile devices, PDF documents, copiers, and printers.
